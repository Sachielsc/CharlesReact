How to update node and npm on windows?
  1. Run PowerShell as Administrator
  2. Type these code: 
    npm install -g npm-windows-upgrade
npm-windows-upgrade
  3. Reference:
    https://stackoverflow.com/questions/18412129/how-do-i-update-node-and-npm-on-windows
